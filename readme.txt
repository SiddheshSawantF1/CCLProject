Steps for installation:
        Download all the files from repository.
        Create a new folder in htdocs folder in XAMPP and unzip all files.
        
Steps to install database:
        Open any browser.
        Go to `localhost/phpmyadmin/`.
        Select Export option and select file named  `bookshack.sql`.
        
Run the website:
        Open any browser.
        Type `localhost/BookShack/`.
        Website is ready to use.
        Sign Up/Sign In to access all features of website.
        
Installation of XAMPP:
        Open any browser.
        Go to `https://www.apachefriends.org/`.
        Follow the download instructions.
